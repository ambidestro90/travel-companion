# Travel Companion

App personale, installabile e funzionante offline, per organizzare più viaggi.

## Versione 0.1.0
- PWA installabile
- struttura predisposta per più viaggi
- primo viaggio: Sharm el-Sheikh 2026
- voli, hotel, parcheggio, assistenza
- libretti assicurativi offline
- checklist con salvataggio locale

## Pubblicazione GitHub Pages
1. Caricare nella radice del repository tutti i file e le cartelle di questo pacchetto.
2. Aprire `Settings` → `Pages`.
3. In `Build and deployment`, scegliere `Deploy from a branch`.
4. Selezionare `main` e `/ (root)`, quindi `Save`.
5. Attendere il completamento della pubblicazione.

## Aggiornamenti
Il file `data/trips.json` conterrà progressivamente tutti i viaggi.
